resvg wheels
============

This repository is used to build and publish precompiled Python
`wheels <https://pythonwheels.com/>`__ for the
`resvg <https://github.com/RazrFalcon/resvg>`__ command-line tool, using
`maturin <https://maturin.rs/>`__.
